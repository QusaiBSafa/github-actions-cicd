# GitHub Actions CI/CD Example

## Description
This is a simple Express API built with TypeScript that demonstrates how to set up a CI/CD pipeline using GitHub Actions.

## What's Built
- A basic Express API with two endpoints: `/api/hello` and `/api/goodbye`.

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the server.
4. Visit `http://localhost:3000/api/hello` or `http://localhost:3000/api/goodbye`.

## API Documentation
- `GET /api/hello`: Returns a greeting message.
- `GET /api/goodbye`: Returns a farewell message.

## Environment Variables
No environment variables are required for this app.
