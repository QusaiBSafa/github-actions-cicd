# Infrastructure Overview
This application is a simple Express API that serves two endpoints.

## Data Models
- No database is used in this example; responses are hardcoded.

## API Design
- `GET /api/hello`: Returns a JSON object with a greeting.
- `GET /api/goodbye`: Returns a JSON object with a farewell message.

## Key Decisions
- Chose Express for its simplicity and popularity in building APIs.
- TypeScript is used for type safety and better development experience.
